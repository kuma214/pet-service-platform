# 개인 맞춤형 반려동물 통합 서비스 플랫폼

## 프로젝트 개요
반려동물 보호자들을 위한 개인 맞춤형 건강, 쇼핑, 커뮤니티 통합 서비스 플랫폼입니다.  
2024년 기업 연계 프로젝트에서 4인이 팀을 이루어 개발했습니다.
